# BeSpider
A spider solitaire clone for the [Haiku Operating System](https://www.haiku-os.org/).

## Usage
To compile, simplyâ€¦

1. Open a Terminal window.
2. `cd` into the `BeSpider-master` folder.
3. Run `make`.

5. Run `open BeSpider`.
6. Have fun!
